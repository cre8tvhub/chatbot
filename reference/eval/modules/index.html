makeEvaluateConversationQuality​

▸ makeEvaluateConversationQuality(«destructured»): EvaluateQualityFunc

Construct a a EvaluateQualityFunc that evaluates the quality of a conversation using an OpenAI ChatGPT LLM.

The returned EvalResult has the following properties:

  • In EvalResult.result, 1 if the conversation meets quality standards and 0 if it does not.
  • In EvalResult.metadata, reason for the result, as generated by the LLM.
